## 📌 About this project

The Jenny-Mod-Jar provides a simple way to add the Jenny character and associated features to Minecraft Java Edition. This mod expands the game world by adding the NPC and supporting logic. It functions across various versions of the game, including recent releases. The project aims to reduce the time spent moving files or editing system paths.

## 🛠️ System requirements

Before you install this mod, ensure your computer meets these basic requirements:

- Operating System: Windows 10 or Windows 11.
- RAM: At least 4GB of allocated memory for Minecraft.
- Java: You must have Java Runtime Environment (JRE) installed. Verify you use the version compatible with your selected Minecraft version.
- Minecraft: You need a legitimate copy of Minecraft Java Edition installed and running on your system.
- Forge: This mod requires Minecraft Forge. Forge serves as the engine that allows external modifications to connect with the base game.

## ⚙️ Installation steps

Installation requires placing the file into your game folder. Follow these instructions closely.

1. Close Minecraft if you have it open.
2. Press the Windows key + R on your keyboard.
3. Type %appdata% into the box and press Enter.
4. Open the folder named .minecraft.
5. Look for a folder named mods. If this folder does not exist, right-click in the empty space, select New, then select Folder, and name it mods.
6. Copy the Jenny-Mod-Jar file you downloaded earlier.
7. Paste this file into the mods folder.

## 🚀 Running the game

Once you place the file correctly, you must launch the game using the correct profile.

1. Open the Minecraft Launcher.
2. Look at the bottom left corner near the Play button.
3. Use the dropdown menu to select your Forge profile. If you do not see a Forge profile, you must install Forge separately from the official Forge website.
4. Click Play.
5. The game will load the modded environment. This might take longer than a standard launch because the software must inject the new code into the game engine.
6. Once the main menu appears, click the Mods button to verify the Jenny Mod is in the list.

## 📂 Troubleshooting common issues

If the mod does not appear in your game, check these common points.

### Incorrect Forge version
Every mod requires a specific version of Forge. If you play on version 1.12.2, you must use the Forge installer for 1.12.2. Mismatched versions prevent the game from recognizing the mod file. Check your Forge installation status if the mod remains hidden.

### File placement
Ensure the file sits inside the /mods/ folder. Placing the file in the .minecraft root folder or inside a different subfolder will stop the game from loading it.

### Java conflicts
Minecraft Java Edition runs on Java. Outdated versions of Java can cause the game to crash during the load screen. Keep your Java installation updated through the official Oracle or Adoptium websites.

### Multiple mod conflicts
Some mods do not work well together. If you have many mods inside your folder, move them to a different location. Test the Jenny Mod by itself. If it works, add your other mods back one by one to find the source of the conflict.

Modded Minecraft consumes more system resources than the base game. Expect longer load times and slightly higher memory usage. If you experience stuttering, lower your game settings, such as render distance or smooth lighting, in the Options menu. This frees up memory for the game to process the new data provided by the mod.
